# AFont
This is AFont, a tool to make fonts from your own handwriting!
Made by Curious Adhvik.

# How to make a Font

**Step 1: Making the Spritesheet**  
  
You will need to make a spritesheet, to make a font. To do this, you need:
* A drawing app. (it needs to have a grid, and the ability to export.)
* A hand.
* The ability to write.

If the app needs to be free, I recommend using *Ibis Paint X*.
Otherwise, you can use *Procreate* for $13.
