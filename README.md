# AFont
This is AFont, a tool to make fonts from your own handwriting!
Made by Curious Adhvik.

# Installation Instructions
1) Install Git and Python ver. 3.14.2 if you haven't already.
2) Run this code in your terminal:
```bash
git clone https://github.com/AdhvikJ/AFont.git
cd AFont
```
3) Create a virtual environment using this code:
``` bash
python3 -m venv .venv
```
4) Activate the venv using this code if you're on Windows:
``` bash
.venv\Scripts\activate
```
and this code if you're on Mac/Linux:
``` bash
source .venv/bin/activate
```
5) Install the required dependencies using this code (universal):
``` bash
pip install -r requirements.txt
```
6) Finally, you can run AFont! (you can use this code)
``` bash
python AFont.py
```
